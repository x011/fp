Flash Player Tutorial

Install Flash Player on ANY Phone or Tablet

  • Step 1 - Enable "Unknown Sources"

    For Android 4 and 5 go to:
    -- Settings -> Security and enable Unknown Sources

    For Android 2 and 3:
    -- Settings -> Application and enable Unknown Sources

  • Step 2 - Download Plash Player

    Navigate to
    www.android.com.de/flash-player
    and Download Adobe Flash Player for Android

  • Step 3 - Install Plash Player

    Tap on the flash player file (install_flash_player.apk) to install it, when finished, tap DONE

  • Step 4 - Install Firefox for Android

    Flash works great with Firefox for Android, install it from the Google Play Store, alternatively, you can install the Dolphin Browser also from the Play Store
