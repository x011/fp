Flash Tutorial

Install Flash on ANY Phone or Tablet

  • Step 1 - Enable "Unknown Sources"

    For Android 4 and 5 go to:
    -- Settings -> Security and enable Unknown Sources

    For Android 2 and 3:
    -- Settings -> Application and enable Unknown Sources

  • Step 2 - Download Flash Player

    Navigate to
    www.android.com.de/flash-player
    and Download Adobe Flash Player for Android

  • Step 3 - Install Flash Player

    Tap on the flash player file (install_flash_player.apk) to install it, when finished, tap DONE

  • Step 4 - Install Firefox for Android

    Flash works great with Firefox for Android, install it from the Google Play Store, alternatively, you can install the Dolphin Browser also from the Play Store

  • Step 5 - Test Flash Player

    Using Firefox or Dolphin Browser Navigate to any website that uses Flash, for ex: www.adobeflashplayer.mobi If you run into problems, make sure that you read the TIPS below.

  • TIPS

    - Clean the temporary files and cookies of your browser and restart the device.

    - Make sure you request the desktop site of the page you're in, some websites automatically deny flash content to mobile devices. To request the desktop site on FIREFOX navigate to the page you want, tap the menu button and choose "request desktop site".

    - Flash Player isn't an APP, it's a plugin that works along with web browsers or apps that use flash technology. You don't need to do anything else after installing it, just navigate to any page that uses flash player an it will run automatically.

    - Depending on your browser configuration, you may have to tap or zoom in/out to activate the flash plugin. You can change this behavior on you browser settings.

    - Confirm that you have JavaScript enabled on your browser settings, some flash content requires JavaScript.

    - Flash Player doesn't work on Chrome Browser. Use it with the Firefox for Android or Dolphin Browser available from Google Play Store.

    - Follow the steps and tips above and you'll have flash player running on your android.
